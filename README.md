# 马坡村头大喇叭 AI 新闻版 📢

一个优雅的 AI 新闻聚合平台，采用老钱风配色设计。

## 🎯 特性

- 📰 聚合 6 个优质中英文 AI 新闻源
- 🎨 老钱风配色（米白、香槟、棕色）
- 🔍 实时搜索和智能筛选
- ⏰ 自动定时刷新
- 📱 响应式设计，支持移动端
- 🕒 只显示 30 天内的新闻

## 📰 新闻源

### 中文源
- **机器之心** - AI 前沿研究
- **36氪** - AI 创业动态
- **晚点LatePost** - 深度商业报道

### 英文源
- **TechCrunch** - 全球科技资讯
- **MIT Technology Review** - MIT 技术评论
- **The Verge** - 科技新闻

## 🚀 本地运行

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动后端

```bash
python ai_news_backend.py
```

后端将运行在 `http://localhost:5001`

### 访问前端

在浏览器中打开 `index.html` 或访问部署的网址。
